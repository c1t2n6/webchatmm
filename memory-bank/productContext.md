# Product Context: Mapmo.vn - Anonymous Web Chat

## Problem Statement
In today's digital age, people struggle to form meaningful connections online. Traditional dating apps focus on superficial appearances and immediate judgments, while social media platforms lack the intimacy needed for genuine conversations. Users need a platform that:

- **Facilitates genuine connections** without the pressure of immediate visual judgment
- **Encourages meaningful conversations** through anonymous, interest-based matching
- **Provides multiple communication modes** (text and voice) for different comfort levels
- **Creates anticipation and excitement** through progressive photo revelation
- **Ensures mutual consent** before revealing personal information

## Target Users

### Primary Users
- **Young Adults (18-35)**: Seeking meaningful connections and relationships
- **Professionals**: Looking for serious relationships or casual conversations
- **Introverts**: Prefer text-based communication before voice/video
- **Privacy-Conscious Users**: Value anonymous communication initially

### Secondary Users
- **Social Explorers**: Want to meet new people with shared interests
- **Relationship Seekers**: Looking for different types of connections (casual, serious, marriage)
- **Curiosity-Driven Users**: Interested in the unique like/reveal system

### User Personas

#### Persona 1: "The Cautious Connector" (25-30)
- **Profile**: Professional, values privacy, prefers text first
- **Needs**: Safe way to meet people, gradual trust building
- **Pain Points**: Rushed judgments, superficial interactions
- **Solution**: Anonymous chat with progressive photo reveal

#### Persona 2: "The Social Explorer" (22-28)
- **Profile**: Outgoing, curious, enjoys meeting new people
- **Needs**: Variety in connections, different conversation types
- **Pain Points**: Boring conversations, lack of meaningful topics
- **Solution**: Interest-based matching with voice call options

#### Persona 3: "The Relationship Seeker" (28-35)
- **Profile**: Mature, looking for serious connections
- **Needs**: Quality over quantity, meaningful conversations
- **Pain Points**: Time-wasting interactions, unclear intentions
- **Solution**: Need-based matching with clear relationship goals

## Core User Journeys

### 1. First-Time User Experience
```
Landing Page → Login/Signup → Profile Setup → Waiting Room → Match → Chat/Call → Like System → Photo Reveal
```

**Detailed Flow:**
1. **Landing Page**: User sees video background with interactive elements
2. **Authentication**: Google Sign-in or username/password registration
3. **Profile Setup**: Complete 4-step profile (basic info, preferences, needs, interests)
4. **Waiting Room**: Choose Chat or Voice Call mode
5. **Matching**: System finds compatible user based on preferences
6. **Communication**: Start anonymous chat or voice call
7. **Like System**: After 5 minutes, decide if they like the person
8. **Photo Reveal**: If mutual like, photos revealed gradually
9. **Continued Connection**: Option to keep conversation or end

### 2. Returning User Experience
```
Login → Waiting Room → Quick Match → Resume Communication
```

**Optimized Flow:**
- Skip profile setup (already completed)
- Direct access to waiting room
- Faster matching with existing preferences
- Resume previous conversations if available

### 3. Voice Call Entry Mode
```
Landing Page → Voice Call → Immediate Matching → Direct Voice Call
```

**Special Flow:**
- User chooses voice call from landing page
- System prioritizes voice call users
- Direct transition to voice call interface
- No text chat step required

## Key Features & Benefits

### 1. Anonymous Communication
**Features:**
- No photos shown initially
- Username-based identification only
- Gradual photo revelation system

**Benefits:**
- Reduces superficial judgments
- Encourages personality-based connections
- Creates anticipation and excitement
- Builds trust gradually

### 2. Smart Matching System
**Features:**
- Gender preference matching
- Interest-based compatibility
- Need-based relationship goals
- Fallback random matching

**Benefits:**
- Higher quality matches
- Reduced time-wasting
- Better conversation starters
- More meaningful connections

### 3. Progressive Photo Reveal
**Features:**
- 3-level revelation: Blur → Semi → Full
- Mutual consent required
- Timer-based like system

**Benefits:**
- Creates excitement and anticipation
- Ensures mutual interest before revealing
- Prevents one-sided attraction
- Builds emotional investment

### 4. Dual Communication Modes
**Features:**
- Text chat for initial connection
- Voice calls for deeper conversation
- Seamless mode switching

**Benefits:**
- Accommodates different comfort levels
- Allows gradual relationship building
- Provides variety in communication
- Matches user preferences

### 5. Like System & Auto-end
**Features:**
- 5-minute timer for like decisions
- Second chance after 5 more minutes
- Auto-end if no mutual interest
- Keep option for continued conversation

**Benefits:**
- Prevents endless conversations
- Encourages active participation
- Respects both users' time
- Creates urgency and engagement
